# LED1000 Mock Knowledge Dataset

Mục đích: bộ dữ liệu giả lập thực tế để kiểm thử chatbot LED1000 trước khi khách cung cấp dữ liệu chính thức.

Quan trọng:

- Đây là dữ liệu demo/mock, không phải bảng giá, tồn kho, chính sách chính thức của LED1000.
- Không dùng bộ dữ liệu này để báo giá production.
- Sau khi khách cung cấp file thật, hãy xóa hoặc tắt các entry mock trước khi import dữ liệu chính thức.

## Câu hỏi test sau import

- LED1000 bán những nhóm sản phẩm nào?
- Adapter 12V 5A giá lẻ bao nhiêu?
- Nếu tôi là khách công trình mua nguồn 12V 20A thì giá dự kiến thế nào?
- Còn hàng LED dây COB 12V 5m không?
- Nếu hết LED dây COB 24V thì có sản phẩm thay thế không?
- Tôi cần LED dây ngoài trời cho bảng hiệu 8m, cần cung cấp thông tin gì?
- Chính sách bảo hành nguồn LED thế nào?
- Xuất VAT cần gửi thông tin gì?
- Tôi là cửa hàng muốn lấy giá đại lý thì cần điều kiện gì?
- Tôi muốn kết nối thêm Zalo/Shopee thì đang có thông tin tài khoản demo nào?
